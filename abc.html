    SELECT * FROM SH.CUSTOMERS;
  2. To see distinct data from the table, run the following statement:
    SELECT DISTINCT * FROM SH.CUSTOMERS;
  3. Find the COUNT
    SELECT COUNT(*) from SH.CUSTOMERS;

%script
